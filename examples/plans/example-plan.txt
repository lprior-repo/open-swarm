# Example Project Plan - User Authentication System

1. Setup database schema [P0]
Description: Create users table with email, password_hash, and metadata fields

2. Implement password hashing [P0]
Description: Use bcrypt for secure password hashing

3. Create authentication middleware [P1] (depends on: 2)
Description: JWT-based authentication middleware for API requests

4. Build login endpoint [P1] (depends on: 2)
Description: POST /api/auth/login endpoint that validates credentials and returns JWT

5. Build registration endpoint [P1] (depends on: 1, 2)
Description: POST /api/auth/register endpoint for new user signup

6. Add password reset functionality [P2] (depends on: 4)
Description: Email-based password reset flow

7. Write integration tests [P2] (depends on: 4, 5)
Description: End-to-end tests for authentication flows
